== Description ==

The best Norwegian weather forecast widget plugin!

Vaeret1.com offers an accurate weather forecast for all cities around the world.
Just set the city and country and embed the widget in your website/blog.

* Widgets are free of charge
* Widget style can be customized
* Responsive weather widget
* No development skills are needed
* High performance and lightweight code
* SEO friendly - no iframes
* Select your display language - Norwegian/English

** For special customization requests please contact this mail -> contact@vaeret1.com

The Vaeret plugin is relying on a 3rd party as a service, the widget is using weather services provided by Vaeret1.com.
* Service provider link - https://www.vaeret.com
* Service provider terms of use link - https://www.vaeret.com/terms
* Service privacy policy link - https://www.vaeret.com/privacy


== Installation ==

1. Upload `vaeret.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the vaeret widget to your page from the widgets tab
